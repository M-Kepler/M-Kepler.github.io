# HTTP 请求耗时

**TTFB**

`Time to First Byte` 也称为首字节时间（TTFB），就是衡量用户的浏览器从你的网站服务器接收第一个 "字节" 的数据所需的时间。不是渲染它或在屏幕上显示它，而是简单地返回一个字节给浏览器。
